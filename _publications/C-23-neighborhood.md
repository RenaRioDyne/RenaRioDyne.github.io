## Definitions
<strong>Simon's congruence:</strong> For an integer $k$,
two strings $w_1$ and $w_2$ are $k$-congruent ($w_1\sim_k w_2$)
if they have the same set of subsequences
of at most $k$.<br>
<strong>The Simon's congruence neighborhood of a language $L$:</strong>
the set of strings that are $k$-congruent to some string in $L$.

## Summary of results
<ul>
<li> PSPACE-completeness results for the following decision problems for two given regular languages:
    <ul>
        <li> Finiteness of the supremum of $k$ that allows
        equivalence of the Simon's congruence neighborhoods </li>
        <li> Inclusion and equivalence of the Simon's congruence neighborhoods for a given $k$ </li>
    </ul>
</li>
<li> Undecidability of computing the maximum $k$ that allows
equivalence of the Simon's congruence neighborhoods for input context-free languages </li>
<li> Decidability of inclusion and equivalence of the Simon's congruence neighborhoods for a given $k$, up to input recursive languages</li>
</ul>
